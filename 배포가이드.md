# 📞 AI 전화 대리인 — 배포 가이드

이 가이드를 따라하면 **30분 안에** 스마트폰에서 바로 쓸 수 있는 앱이 완성됩니다.

---

## 📋 필요한 것 (모두 무료 시작 가능)

| 서비스 | 용도 | 비용 |
|--------|------|------|
| GitHub 계정 | 코드 업로드 | 무료 |
| Render.com | 서버 호스팅 | 무료 |
| Twilio | 전화 발신 | 무료 체험 $15 크레딧 |
| Anthropic | AI 분석 | 종량제 (소량 무료) |

---

## 🚀 STEP 1 — GitHub에 코드 올리기

1. https://github.com 가입 후 로그인
2. 우측 상단 **+** → **New repository**
3. Repository name: `ars-ai-agent`
4. **Create repository** 클릭
5. 이 폴더의 모든 파일을 업로드
   - "uploading an existing file" 클릭
   - 파일 전체 드래그 앤 드롭
   - **Commit changes** 클릭

---

## 🚀 STEP 2 — Render.com에 서버 배포

1. https://render.com 가입 (GitHub 계정으로 로그인 가능)
2. **New** → **Web Service** 클릭
3. GitHub 저장소 `ars-ai-agent` 선택
4. 아래와 같이 설정:
   - **Name**: ars-ai-agent
   - **Runtime**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
5. **Create Web Service** 클릭
6. 배포 완료 후 URL 확인: `https://ars-ai-agent-xxxx.onrender.com`
   - ⚠️ 이 URL을 복사해두세요!

---

## 🚀 STEP 3 — Twilio 설정

1. https://twilio.com 가입
2. Console 메인에서 **Account SID**와 **Auth Token** 복사
3. 좌측 메뉴 **Phone Numbers** → **Buy a number**
   - 한국(+82) 번호 또는 미국(+1) 번호 구매
   - 미국 번호가 저렴 (월 $1)
4. 구입한 번호 복사 (예: +14155552671)

---

## 🚀 STEP 4 — Anthropic API 키 발급

1. https://console.anthropic.com 가입
2. **API Keys** → **Create Key**
3. 키 복사 (sk-ant-로 시작)

---

## 🚀 STEP 5 — 앱에서 설정 입력

1. 브라우저에서 Render URL 열기
   - `https://ars-ai-agent-xxxx.onrender.com`
2. 우측 상단 **⚙️ 버튼** 클릭
3. 아래 정보 입력:
   - Twilio Account SID
   - Twilio Auth Token
   - Twilio 전화번호 (+82 또는 +1 포함)
   - Anthropic API Key
   - 서버 URL (Render URL 그대로)
4. **저장** 클릭
5. 3개 항목 모두 ✓ 초록색으로 바뀌면 완료!

---

## 🚀 STEP 6 — Twilio 웹훅 설정

Twilio가 전화 응답을 서버로 보낼 수 있도록 설정해야 합니다.

1. Twilio Console → **Phone Numbers** → 구매한 번호 클릭
2. **Voice Configuration** 섹션
3. **A call comes in** → Webhook 선택
4. URL 입력: `https://ars-ai-agent-xxxx.onrender.com/twilio/voice`
5. **Save** 클릭

---

## 📱 사용법

앱을 열고 마이크 버튼을 누른 후 말하세요:

- **"신한은행에 전화해서 잔액 조회해줘"**
- **"국민건강보험공단 보험료 조회해줘"**
- **"SKT 이번달 요금 알려줘"**

또는 하단 빠른 실행 버튼을 탭하세요.

AI가 자동으로:
1. 기관 파악 → 전화번호 검색
2. Twilio로 전화 발신
3. ARS 음성 인식 → 번호 자동 선택
4. 상담원 연결 시 알림음 + 배너 표시
5. 직접 통화 가능

## ❓ 자주 묻는 질문

**Q. Render 무료 플랜은 잠자기(Sleep)가 있다고 들었어요**
A. 무료 플랜은 15분 비활성 시 잠드는데, 첫 요청 시 30초~1분 느릴 수 있습니다.
   유료 플랜($7/월)으로 업그레이드하면 항상 켜져 있습니다.

**Q. Twilio 무료 크레딧으로 얼마나 쓸 수 있나요?**
A. $15 크레딧으로 한국 번호 발신 기준 약 50~100회 정도 사용 가능합니다.

**Q. ARS 메뉴가 바뀌면 어떻게 하나요?**
A. `data/institutions.json`에서 해당 기관의 `services` 항목을 수정 후 GitHub에 다시 올리면 자동 재배포됩니다.
